== Description ==

A Wordpress plugin for publishing and maintaining software projects.

== Installation ==

1. Upload software-hub to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [software_hub_$softwarename] in your wordpress pages where you wish the
hub to appear, replacing $softwarename with the name of the software you want to 
publish a hub page for.
